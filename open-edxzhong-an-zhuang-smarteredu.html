Open edX中安装SmarterEdu

  • by cn
  • on 三 04 三月 2015

Category: edxsmarteredu

以下过程给edX安装SmarterEdu。

安装的位置是edX的cms

1）首先将你的SmarterEdu项目文件夹复制到/edx/app/edxapp/edx-platform/cms目录下，结构如下：

edx-platform/cms/ 
  + djangoapps/
  + envs/
  + lib/
  + smartERproject
      + edx_if/
      + smarteredu/
      + static/
      + templates/
      + manage.py
      + ......
  + static/
  + templates/
  + .coveragerc
  + ......

2）在 /edx/app/edxapp/edx-platform/cms/envs/common.py 中设定 INSTALLED_APPS，以安装SmarterEdu的包：

INSTALLED_APPS = (
    # SmarterEdu
    'smarteredu.course',
    'smarteredu.appauth',
)

3）在 /edx/app/edxapp/edx-platform/cms/envs/common.py 中设定cms的包查找路径：

sys.path.append(REPO_ROOT)
sys.path.append(PROJECT_ROOT / 'djangoapps')
sys.path.append(PROJECT_ROOT / 'smartERproject')  # 添加该设定后，cms就会查找到你的项目
sys.path.append(COMMON_ROOT / 'djangoapps')
sys.path.append(COMMON_ROOT / 'lib')

4）同步edX数据库：

sudo -u www-data /edx/bin/python.edxapp /edx/app/edxapp/edx-platform/manage.py cms --settings aws syncdb
sudo -u www-data /edx/bin/python.edxapp /edx/app/edxapp/edx-platform/manage.py cms --settings aws migrate
